Para realizar a instalaçao do sistema em localhost, basta realizar as seguintes etapas:
1- Instalar o Node.js
2- Após realizar o clone do app ou download
3- Abrir com o CMD a pasta 
4- ainda no cmd digitar o seguinte comando 'npm i' , para instalar as dependencias
5- para rodar a api no mesmo cmd basta digitar 'npm run api'
6- para abrir a aplicaçao basta entrar na pasta do project_board
7- clicar no arquivo index.html

////////////////////////////////////////////////////////////////////

Para rodar os testes automatizados:
1- no vscode instar em extensoes o live server
2- abrir a pagina html no live server
3- entrar na pasta do project_board pelo cmd
4- digitar no cmd npm run teste
5- escolher o teste a ser executado
